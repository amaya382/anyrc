# anyrc example
## Files you need
* `.anyrc`
    * Nomally, you needn't care of this file
        * This file will be installed automatically in the anyrc installation
        * Pre-configured for zsh, bash, tmux, git, and vim
    * If you want to support other tools, edit me
* `.anyrc.d/*`
    * Put your dotfiles here. Symlinks are allowed.
